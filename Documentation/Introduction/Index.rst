============
Introduction
============

`WebP <https://en.wikipedia.org/wiki/WebP>`_ is an image format developed by Google. According to Google's measurements, a conversion from PNG to WebP results in a 45% reduction in file size when starting with PNGs found on the web.

Our own tests with JPEGs revealed reductions in file sizes from about 25% up to more than 95%.

.. _what-it-does:

What does it do?
================

This extensions provides an alternative partial for image rendering with multiple resolutions and support for WebP format.

Furthermore it provides alternative partials for the extension *news*.

And additionally it provides a viewhelper to include SVG images as inline html.
